# environment-protection-server
基于SpringBoot+Vue前后端分离的爱心绿动环保网系统（源码+文档），系统包含管理员、用户角色，功能如下： 管理员：首页统计（报名统计、文章统计）；用户管理；文章分类管理；文章管理；活动管理；活动报名；论坛管理（论坛版块、帖子管理、评论管理）；物品管理；物品兑换；留言管理；公告信息；个人信息；密码修改； 用户：注册、登录系统；首页；环保知识（分类、知识列表、知识详情）；环保活动（活动列表、详情、报名）、环保论坛（版块、帖子列表、详情、点赞、收藏）；积分兑换（物品详情、兑换）；系统留言；公告信息。用户还具有个人中心功能，功能包括：活动报名、我的帖子、帖子评论、帖子点赞、帖子收藏、商品兑换、我的留言、个人中心、修改密码。
